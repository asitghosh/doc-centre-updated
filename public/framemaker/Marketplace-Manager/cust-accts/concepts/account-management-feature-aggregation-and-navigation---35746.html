Account management feature aggregation and navigation

Table 5-13 is a quick reference to all Marketplace Manager features to work with companies and user accounts.

Table 5-13: Feature navigation for company and user management


Manage > Channel > Marketplace >

Home |

Users

Use UI elements and features here, optionally, to work with user account records.

    • Browse, filter, sort, and search the list of all registered users, and export any or all of its underlying records to CSV.
    • View the highly detailed information in any user record.
    • Manage the pending invitations for potential users to join companies on your marketplace.

Companies

Use UI elements and features here, optionally, to work with company records.

    • Browse, filter, sort, and search the list of all registered companies, and export any or all of its underlying records to CSV.
    • Create a new company and invite its first associated user.
    • View the highly detailed information in any company record.
