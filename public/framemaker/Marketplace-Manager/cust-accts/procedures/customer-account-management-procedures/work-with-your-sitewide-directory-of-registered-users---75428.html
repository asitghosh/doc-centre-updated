Work with your sitewide directory of registered users

As a channel administrator, you can search, filter, and view a list of all registered users on your marketplace.

Navigation path

Manage > Channel > Marketplace > Home | Users

Before you begin

    • Log in to your marketplace as its administrator.

Procedure

    1. Step 1. Go to Manage > Channel > Marketplace.

      2. Step 2. Click through Home | Users.

      Step 3. Click Show Filters to expand a toolbar whose controls help you to limit which user accounts the table should show.

      Step 4. Do any combination of the following, until you find the user accounts that matter to you.

NOTE: Selections take effect immediately, and each effect is additive.

    • Do you want to list activated user accounts exclusively?

Cite only the user accounts that are active

      1. a. Choose Only Active from the Show list.
    • Do you want to list inactive user accounts exclusively?

Cite only the user accounts that were never activated

      1. a. Choose Only Inactive from the Show list.
    • Do you want to list only the user accounts that were created within a specified range of dates?

Cite only the user accounts that were created within a specified date range

      1. a. Define the start and end dates by entering them manually in the paired date-entry fields.

        OR
        Click a field’s corresponding calendar icon to expose the date-picker widget for populating that field.

NOTE: When you enter dates manually, use the calendar date notation syntax that is standard in your locale and which your marketplace is configured to use — either M/D/YY or D/M/YY.
    • Do you want to download a CSV file that matches only the records that your filters return?

Save records for offline processing

      1. a. Click Download CSV to save a local copy of the table’s underlying records as you have trimmed them

      Step 5. ( Optional ) Enter a query in the Search field. Then, either press Enter or click the magnifying glass icon.

When your query matches one or more records, we trim the table to expose only these matches.

      Step 6. ( Optional ) Click a table row to show more in-depth details from its underlying record.

      Step 7. Stop. You have completed this procedure.
