Edit the tags that organize your product catalog

Navigation path

Manage > Channel > Products > Groups | Product Groups

Procedure

    1. Step 1. Go to Manage > Channel > Products.

      2. Step 2. Click through Groups | Product Groups.

      Step 3. ( Optional ) Make category tagging mandatory or optional for developers when they integrate new products into your marketplace.

      1. a. Click Edit Category Settings on the Product Groups page.
        The Edit Category Settings popup opens.

        2. b. Check or clear the Required field checkbox.

        c. ( Optional ) Enter tooltip text for your developers.

        d. Click Save.

      Step 4. Then, do one or more of the following.

        • Would you like to add a category?

      Add a category

          1. a. Click Add Another Category on the Product Groups page.

      The Edit Category popup opens.

            b. Enter a category label.

      Then, check or clear any or all of its associated checkboxes:

          • Use for homepage navigation
          • Only Channel Admins can set
          • Use badge for category

            • c. ( Optional ) Enter one or more subcategory labels.
        • Would you like to edit the label for an active category or any of its subcategories?

      Edit a category or its subcategories

          1. a. Click the name of any defined category on the Product Groups page.

      The Edit Category popup opens.

            b. ( Optional ) Edit the category label.

      Then, check or clear any or all of its associated checkboxes:

          • Use for homepage navigation
          • Only Channel Admins can set
          • Use badge for category

            • c. ( Optional ) Edit one or more subcategory labels.

        • Would you like to delete an active category and all of its subcategories?

      Delete a category and all of its subcategories

          1. a. Click the name of any defined category on the Product Groups page.

      The Edit Category popup opens.

            b. Click Delete Category.

      The Confirm Delete prompt asks you, “Are you sure you want to delete this product group? Products will no longer be tagged with this group.”

            c. Click Yes.
        • Would you like to delete one or more of the subcategories from a category?

      Delete subcategories from a category

          1. a. Click the name of any defined category on the Product Groups page.

      The Edit Category popup opens.

            b. Click the Delete button (

        ) to the right of a subcategory label to delete that category.
        • Would you like to create a new subcategory?

      Add a subcategory to a category

          1. a. Click the name of any defined category on the Product Groups page.

      The Edit Category popup opens.

            b. Click Add Another Subcategory.

            c. Enter the subcategory label.

      Step 5. Click Save.

      Step 6. Stop. You have completed this procedure.
