Understand password policies

You might not know precisely which policy — lenient, moderate, or strict — defines your use of passwords on a given marketplace, but you can always give yourself a strict password. All marketplaces accept strict passwords.

NOTE: The password policy on a given marketplace is not immediately relevant when that marketplaceenters stealth mode and employs a sitewide password. In such cases, your browser cannot load any data, including even the standard password prompt, until you authenticate at the higher level. See Access a password-protected server.

Table 1-3: Password policy stipulations

Strict

Each registered user account must be secured with a password that is at least 10 characters long and matches for this pattern:

    • contains at least one uppercase letter ( A - Z )
    • contains at least one lowercase letter ( a - z )
    • contains at least one digit ( 0 - 9 ) or special character ( ~!@#$%^&*()_+=<>? )

In the strict policy, passwords expire automatically and must be changed when they age to 90 days. Depending on how the marketplace is configured, it might notify users through email when their passwords are scheduled to expire.

Moderate

Each registered user account must be secured with a password that is at least 8 characters long and matches for this pattern:

    • contains at least one letter ( A - Z or a - z )
    • contains at least one digit ( 0 - 9 ) or special character ( ~!@#$%^&*()_+=<>? )

In the moderate policy, passwords expire automatically and must be changed when they age to 90 days. Depending on how the marketplace is configured, it might notify users through email when their passwords are scheduled to expire.

Default (Lenient)

Each registered user account must be secured with a password that is at least 8 characters long and matches for this pattern:

    • contains at least one letter ( A - Z or a - z )
    • contains at least one digit ( 0 - 9 ) or special character ( ~!@#$%^&*()_+=<>? )

In the default policy, passwords do not expire automatically.

Understand password expiration behaviors

Some marketplaces expire passwords automatically and others do not. On the marketplaces that do expire passwords automatically:

    • Expiration occurs after 90 days. Nonetheless, any registered user can change his or her password after any shorter interval than 90 days and, thus, reset the expiration countdown.
    • Some will send password expiry notification messages automatically, through email.
