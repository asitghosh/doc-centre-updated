Understand custom fields for companies

Navigation path

Manage > Channel > Settings > Custom UI | Custom Form Fields > ( Company Custom Attributes )

Your marketplace collects and stores certain details whenever a new company is created. When the standard set of collected details is not sufficient for your purposes, you can add UI fields to the company creation workflow.

Later, to access these additional details, you can submit calls to the API endpoint that returns details about one company specifically.

NOTE: When you lack access to this feature because access to it is not turned On, contact your AppDirect account representative.
