Navigation path

Manage > Channel > Settings > Settings | Marketplace Functionality

You can turn a marketplace behavior On or Off that hides the { full name } > My Profile page and the { full name } > My Company page from all users.

    • Features at { full name } > My Profile help users to access and edit identifying details about themselves, such as their first name, last name, and telephone number.
    • Features at { full name } > My Company help users to access simple details about their associated company, such as its mission statement and mailing address.
    • Features at { full name } > My Company > Directory help users to browse and search simple records about the other registered users who share their associated company in common with them.
