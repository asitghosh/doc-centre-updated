Configure custom fields for user account creation

Navigation path

Manage > Channel > Settings > Custom UI | Custom Form Fields > ( User Custom Attributes )

Before you begin

    • Log in to your marketplace as its administrator.

Procedure

    1. Step 1. Go to Manage > Channel > Settings.

      2. Step 2. Click past Custom UI | Custom Form Fields.

      Step 3. Click Add Form Field above the User Custom Attributes table.

The Add Form Field popup opens.

      1. a. Enter a database column heading in the Key field.

Your entry identifies the new attribute type uniquely in backend records and is also the API key.

        b. Enter a concise text fragment in the Label field.

Your entry serves as the UI element label for a field that you are adding to the user account creation form.

        c. Enter a descriptive summary in the Hint field.

Your entry functions like a tooltip, assisting anyone who interacts with the new UI field but doesnâ€™t understand its purpose.

        d. ( OPTIONAL ) Operate the Visible in UI checkbox.

      1. Check the checkbox to ensure that the a UI field is available to capture the custom value. Otherwise, the only way to capture the custom value is through an API call.

        OR
      2. Clear the checkbox to prevent the user account creation UI from showing the field. In this case, the only way to capture the custom value is through an API call.

        3. e. Click Save.

        OR
        Click Cancel.

      Step 4. Stop. You have completed this procedure.
