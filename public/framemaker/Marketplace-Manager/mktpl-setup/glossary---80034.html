B

Bundle

A deprecated product type.

NOTE: Bundle is nomenclature for a kind of product integration that we no longer support. However, we continue to support the bundling together of mutually complementary products for sale, as a merchandising tool that drives marketplace revenue. See bundle.

D

Domain Provider

Internet domain registration service provided through one favored registrar, of the marketplace providerâ€™s choosing, in support of third-level domains that Microsoft encourages its business users to obtain and manage. This product type is of no use on a marketplace that does not sell Microsoft business products.

Download

A product whose consumer must host and use it locally.

Downloadable Bundle

A deprecated product type.

E

event

Any UI-driven or API-driven trigger for a detection-and-notification mechanism that is built in to the AppDirect platform, which can be triggered when:

    • A significant database entity is created, changed, or deleted.
    • Any relationship between two or more such entities is established, changed, or terminated.
    • A system component or integration meets the conditions of a defined metric.
    • A system component or integration fails a test (exceeds a threshold or range) for acceptable operation.

Events are the basis for all:

    • server and system logs
    • records that reports cite
    • notification messages that email templates spawn

External Application

A deprecated product type.

I

Infrastructure Services

This product type uses vendor-proprietary connectors to pull pricing and usage records for hosted infrastructure offerings, such as Amazon AWS, Microsoft Azure, or VMware vCloud.

P

product type

Developers must specify a product type for each product that they request to list in your product catalog. And, before they can do this, you must specify which product types your marketplace should expose to its developer community. The defined types are as follows:

    • Bundle
    • Domain Provider
    • Download
    • Downloadable Bundle
    • External Application
    • Infrastructure Services
    • Stacked Editions
    • Standing Cloud
    • Static
    • Training Video
    • Web App
    • Web Application (manual setup)

S

Stacked Editions

See stacked edition

Standing Cloud

A product type designation for private applications that use an AppDirect-proprietary connector to send and receive event notifications.

Static

A deprecated product type.

T

Training Video

A product type designation for video content that educates your customers about a related product, which your marketplace sells separately. This product type limits vendors to a minimal product marketing profile, few pricing options, and no SSO into any web service.

W

Web App

This product type is the one that marketplaces expose most often to developers, and that developers choose most often for their products. It grants access to the full and standard range of features for marketing, billing, integration, and management.

Web Application (manual setup)

Instead of relying on API integrations, like Web Apps do, this product type notifies channel administrators through email to record customer provisioning events manually. This type imposes a PENDING state on customer transactions that could otherwise take effect in real time. It grants access to the full and standard features for marketing, but limits whatâ€™s possible for billing, integration, and management.
